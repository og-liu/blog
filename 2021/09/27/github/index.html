GitHub 无法访问解决方案

发表于2021-09-27|更新于2021-09-27

第一步: 查看 GitHub 的 IP 地址

进入网址 https://github.com.ipaddress.com

第二步: 查看 GitHub 的 域名 IP

进入网址 https://fastly.net.ipaddress.com/github.global.ssl.fastly.net

第三步: 查看 GitHub 的 静态资源 IP

进入网址 https://github.com.ipaddress.com/assets-cdn.github.com

第四步: 修改 Hosts 配置文件

将上面三步所得的 IP 写进 hosts 文件里

注意事项

  1. 这些 IP 过一段时间会变动，所以网上很多教程都是直接复制第四步，却没有效果，这时候一定记得要查一下 IP 是否一样。
  2. 如果在浏览器开着 GitHub （无法访问状态）的时候修改了 hosts 文件需要重启浏览器。

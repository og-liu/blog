个人博客搭建与部署 (一)

发表于2020-12-05|更新于2020-12-29|教程

前言篇

本系列文章将完整的讲述如何快速搭建并部署上线一个博客系统, 希望能帮助到有需要的人, 文笔不好, 技术含量粗浅, 老鸟勿喷，欢迎指正与交流 ！

目标

博客系统, 部署至 Github Pages 以及阿里云ESC服务器, 并通过安全(https)域名访问.

需要用到的前提准备（掏钱买）

  • 阿里云ESC服务器一台, 新人首购折扣非常低, 建议直接买三年.
  • 阿里云域名一个, 注册比较麻烦, 价格不一, 常规 .com 域名约为60多软妹币一年.

知识储备

本系列文章将完全面向小白的视角编写, 所用到的技术点都会详细解释, 所以基本不需要什么知识储备, 能够理解网站建设的基本概念即可.

将会使用到的相关内容

  • Hexo 快速、简洁且高效的博客框架
  • Hexo 风格主题
  • Travis CI 项目部署工具
  • Valine 评论系统
  • Gravatar 头像图片管理
  • LeanCloud 云服务，用于管理评论数据
  • Github 代码托管平台
  • Github Pages 由Github提供的站点
  • ssh、node/npm、git、ubuntu、nginx…
